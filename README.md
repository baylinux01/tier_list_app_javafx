# tier_list_app_javafx
Tier List yaparken kullanılabilecek program. Resim ve altına yazı destekliyor. Sürükle bırak ile çalışıyor oldukça pratik.
Kullandığım javafx bağımlılıklarının sürümünden dolayı uygulama sadece Java25 ve üstü ile çalıştırılınca çalışıyor.
Çalıştırırken Java25'in devrede olduğundan emin olmak için java binary'sinin tam yolu ile çalıştırmak gerekebilir.
Uygulama derlendikten sonra java 25 ve üstü yüklü ise linux terminalinde 
/home/tam/yol/to/java25/binary/bin/java -jar tierList.jar
kodu ile çalıştırılabilir.
Uygulama derlenmeden eclipse ile de çalıştırılabilir ama yine Java25 runtime ile çalıştırılması için bazı ek ayarlamalar gerekebilir.
Ya da Java25 yoksa javafx dependencyleri daha eski bir sürüme çekilerek uygulama yine çalıştırılabilir.
